Sistema de Gestión Educativa

Spring EduManager es una plataforma desarrollada con Spring Boot que permite:

  • Gestionar estudiantes con validación de datos
  • Administrar cursos y sus características
  • Controlar acceso mediante autenticación JWT
  • Acceder a APIs REST para integración

Desarrollado como parte del Bootcamp
